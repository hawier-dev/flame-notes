# Flame notes
### Note app project build with [Django](https://www.djangoproject.com/).
___________________
Start app with: 
```
git clone https://github.com/hawier-dev/flame-notes.git
cd flame-notes
pip install -r requirements.txt
python manage.py runserver
```
